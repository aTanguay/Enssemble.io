# SAM2695 GM Voicing Reference
### All 128 General MIDI Programs + MT-32 Bank

Use Program Change (PC) messages to select voices. Send CC 0 = 0 for GM bank, CC 0 = 127 for MT-32 bank, followed by PC message.

## Bank 127 — MT-32
*Access via CC 0 = 127, then Program Change*

The MT-32 bank provides Roland MT-32 compatible sounds — a different character
from GM. More vintage, more '80s/'90s electronic. Great for leads and textures.
Notable differences from GM: different piano voicing, distinctive synth leads,
and unique atmospheric pads not in the GM set.

*Note: Program numbers map to the same slots 0–127 but with different timbres.*

## Quick Reference — Button Navigation

```
[0] short  →  Next curated preset for this unit's role
[1] short  →  Previous curated preset
[2] short  →  Next GM instrument (+1 fine)
[3] short  →  Previous GM instrument (-1 fine)
[0] long   →  Next GM category (+8 jump)
[1] long   →  Previous GM category (-8 jump)
[2] long   →  Volume up
[3] long   →  Volume down
```

*Changing voice plays a short audition note so you always know what you're hearing.*
